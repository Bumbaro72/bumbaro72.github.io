About

We are awesome TEAM

Hi. I'm rationalirl on twitch and I'm developing Free Software for robust high quality video live streaming using mobile 4G/5G networks under the BELABOX project.

The current BELABOX implementation is basically a proof-of-concept minimum viable product. My TODO list includes things such as:


Join

  • making the whole installation and configuration process easier
  • developing support for alternative / cheaper / more powerful hardware
  • extending the functionality with features requested by current and prospective users
  • nd of course I'm always looking for ways to improve the video quality and reliability


This started as a passion project, and so far it's been developed entirely on my own time and using my own money to purchase the required hardware.
As the project is growing outside the scope I've initially envisioned, I'm considering shifting some more time to it and away from paying gigs.
Your donations will help me spend more time working on BELABOX and providing support.


Starting with the $5 tier, sponsors get a BELABOX cloud remote account and starting with the $10 tier, sponsors get access to my
BELABOX cloud SRT + srtla relay servers. These allow you to pull either a BELABOX (srtla bonded) stream or a phone SRT stream into OBS
without installing any other software on your home computer and without running a VPS.

The relays run on servers hosted in datacenters and use my custom SRT server software which outputs stats compatible with NOALBS automatic scene switching.

All sponsor perks are now self-managed via https://cloud.belabox.net

Do you work for a company using or interested in using any of the BELABOX tech stack? I'm available for consulting work, e.g. porting BELABOX to new hardware platforms,
developing new features on request, tweaking the software for specific use cases, general support. Contact me at belaboxproject [at] gmail [dot] com.
